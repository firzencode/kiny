## [Unreleased]

## [0.4.0] - 2026-06-28
### Added

- editor 接入 AI：配置自己的 LLM（OpenAI 兼容：OpenAI / DeepSeek / GLM / Ollama 等），在 AI 面板里让 AI 帮你建 / 改节点，改动落进可撤销的编辑缓冲；key 仅存本机、直连不中转。
- editor 内置 AI 现可查阅 Kin 语言规范：AI 拿不准语法时按章节查规则 / 示例 / 边界（以语言规范为唯一来源），写出的 Kin 更准。
- editor 自动保存与崩溃恢复：未保存改动会在后台写入恢复草稿（不碰真文件），崩溃或强制退出后重开项目可一键恢复；设置里可开关「自动恢复草稿」（默认开）。

### Fixed

- 修复遇到「无玩家交互的死循环」（如漏写选项标记致无限自动跳转）时界面完全卡死：现改为报错并指出出错节点，editor / reader 正常显示错误而非冻结。同时 editor 编辑期对「漏写 `*` / `+` 选项标记、被当成正文+跳转」的写法给出告警。

## [0.3.0] - 2026-06-26
### Added

- reader 支持存档读档：自动续读（读一半重开从断点继续）+ 手动多存档（存 / 读 / 删，删除需二次确认、存档有「已存档」提示）。

### Changed

- editor 编辑内核迁移到 CodeMirror 6：查找 / 替换、多光标、代码折叠、括号匹配、行内诊断波浪线，以及节点名 / 变量 / @命令自动补全与节点跳转（F12 / Ctrl+点击）。

## [0.2.0] - 2026-06-24
### Added

- 文本支持内联富文本：粗体 / 斜体 / 下划线 / 删除线、颜色、字号与换行。
- editor 跟进内联富文本：语法参考新增「内联富文本」一节，编辑器高亮识别富文本标签。
- editor / reader 运行时错误收集与问题反馈：错误落本地日志、崩溃有兜底页，帮助菜单可一键反馈。
- editor 可把项目导出成自包含独立网页：file:// 双击即读完整故事（含图片 / 音频），不依赖联网。

### Changed

- editor 设置弹窗字号 / 行距步进器视觉优化：px 单位独立分格、数字不再跳动。

## [0.1.0] - 2026-06-21
### Added

- 实现 Kiny DSL 引擎:parser(文本→AST)、analyze(跨文件静态检查)、runtime(状态机执行)与 CLI 终端播放器。
- 支持完整叙事语法:节点/子节点、选项(一次性/粘性/条件/fallback)、`>` 分支汇合、`<>` 粘连、文本变体(`seq`/`cycle`/`once`/`shuffle`)、`@if` 条件块、内联 JS 求值、带参节点、回合数与可复现随机。
- 提供平台无关的 React 播放层 player:受控 `<Player>` 组件、driver/host 抽象、确定性保位重放。
- 支持背景、音频与一次性音效(`@sfx`)命令。
- 实现故事状态快照:`Story.serialize` / `restoreStory` 往返等价。
- 新增浏览器端故事阅读器 web-reader(Vite + React)。
- 新增 Tauri 2 桌面编辑器 editor:多文件项目、文件树增删改/拖拽、跨文件实时校验、保位预览、多 tab 与会话恢复、语义着色、排版与主题设置、`.kip` 故事包导出、未保存守卫。
- 新增 Tauri 2 桌面阅读器 reader:`.kip` 导入(按钮/拖放)、持久书架与阅读屏。
- 内置样例项目《雾港之夜》。
